# 合理餐食组合（供参考）

## 500千卡午餐组合
- 鸡胸肉 100g（165千卡） + 糙米饭 150g（174千卡） + 西兰花 200g（68千卡） + 橄榄油 5g（45千卡） → 总计约452千卡
- 三文鱼 120g（208千卡） + 烤土豆 150g（122千卡） + 芦笋 200g（40千卡） + 酸奶酱 30g（30千卡） → 总计约400千卡
- 豆腐 200g（152千卡） + 荞麦面 100g（140千卡） + 炒青菜 200g（50千卡） + 鸡蛋 1个（72千卡） → 总计约414千卡

## 300千卡早餐组合
- 全麦面包 2片（约180千卡） + 牛奶 200ml（122千卡） → 总计302千卡
- 燕麦片 40g（147千卡） + 香蕉 1根（105千卡） + 酸奶 100g（72千卡） → 总计324千卡
- 煮鸡蛋 2个（144千卡） + 橙子 1个（47千卡） + 黑咖啡 → 总计191千卡（可增加一片全麦面包）

## 400千卡晚餐组合
- 蒸鱼 150g（170千卡） + 糙米 100g（116千卡） + 凉拌黄瓜 200g（30千卡） + 豆腐汤 1碗（80千卡） → 总计396千卡
- 鸡腿（去皮）150g（198千卡） + 烤红薯 150g（129千卡） + 清炒菠菜 200g（46千卡） → 总计373千卡
